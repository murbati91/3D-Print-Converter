## Flashing the Firmware

### Using PlatformIO (Recommended)

```bash
# Install PlatformIO CLI
pip install platformio

# Navigate to firmware folder
cd 3d-print-converter/firmware

# Build and upload
pio run -t upload

# Monitor serial output
pio device monitor
```

### Using Arduino IDE

1. Install ESP32 board support
2. Select "ESP32 Dev Module"
3. Install libraries:
   - TFT_eSPI
   - ArduinoJson
4. Copy pin defines from platformio.ini to User_Setup.h
5. Upload

---

## First Boot

1. **Power on** via USB
2. **Wait** for AP mode (no WiFi configured)
3. **Connect** to WiFi: "3DConverter" / Password: "12345678"
4. **Open** http://192.168.4.1
5. **Configure** your home WiFi
6. **Reboot** - it will connect to your network

## Troubleshooting

### TFT Shows White/Nothing
- Check all 8 wires
- Verify 3.3V power (not 5V!)
- Try swapping MOSI/MISO

### SD Card Not Detected
- Format as FAT32
- Check CS pin connection
- Try different SD card

### Can't Upload Firmware
- Hold BOOT button while pressing EN
- Release both
- Try upload again

### WiFi Not Connecting
- Check SSID/password (case sensitive)
- Move closer to router
- Try AP mode first

## Need Help?

The screw terminals make this super easy - no soldering required!
Just strip wire ends and screw them in.

**Wire Gauge**: 22-26 AWG works best with these terminals.
