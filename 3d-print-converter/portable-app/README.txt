============================================
   3D Print Converter - Portable App
   by Tech Sierra Solutions
============================================

Convert CAD files (PDF, DWG, DXF, SVG) to
G-code for 3D printing.

REQUIREMENTS:
-------------
1. Windows 10/11
2. Python 3.11 or higher
3. Internet (for first-time setup only)

OPTIONAL (for full conversion support):
- Inkscape (PDF conversion)
- PrusaSlicer (G-code generation)
- ODA File Converter (DWG files)

FIRST TIME SETUP:
-----------------
1. Install Python from https://python.org
   (Check "Add Python to PATH"!)

2. Run "Install-Dependencies.bat"

3. Run "Start-3D-Converter.bat"


HOW TO USE:
-----------
1. Double-click "Start-3D-Converter.bat"
2. The converter app opens in your browser
3. Drag & drop files or click to upload
4. Adjust settings (scale, format, etc.)
5. Converted files download automatically

SUPPORTED FORMATS:
------------------
Input:  PDF, DWG, DXF, SVG, DAT
Output: G-code, STL, OBJ, 3MF


TROUBLESHOOTING:
----------------
- "Python not found": Install Python and
  check "Add to PATH"

- "Connection failed": Make sure the server
  window is running (minimize, don't close)

- "Conversion failed": Check if required
  tools (Inkscape, PrusaSlicer) are installed
